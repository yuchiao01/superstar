張凌赫，本名張家瑋，中國內地男演員。畢業於南京師範大學電氣工程專業，是名副其實的「理工男神」。 2019年主演古裝劇《少女大人》正式進入演藝圈。 2022年憑藉古裝仙俠劇《蒼蘭訣》中飾演長珩一角獲得廣泛關注。 2023年主演的《雲之羽》、《寧安如夢》更讓他人氣大漲，成為新生代最受矚目的男演員之一。
