關於寧哥

劉宇寧是中國內地流行樂男歌手、影視演員，摩登兄弟樂隊主唱。2014年組建樂隊摩登兄弟並擔任主唱，通過YY直播、抖音平台演唱流行歌曲而走紅。

2018年正式個人出道後，開始活躍於歌唱舞台與綜藝節目，陸續推出多首個人單曲、EP與專輯。首張個人專輯《十》獲得2020年東方風雲榜傳媒推薦專輯獎。

近年更成功轉型為實力派演員，主演《一念關山》《紫川》《折腰》《珠簾玉幕》等多部熱播劇，憑藉出色的演技獲得廣泛好評。
